Installation

To play this level, extract the file "sands.bsp" into
your sierra\half-life\valve\maps directory and then start the game.
Start a deathmatch server and type "map sands" or "changelevel
sands" in the console. To test it out in single player mode
start Halflife with hl.exe -console -toconsole -dev. This enables
you to try some stuff using the cheat codes.
