Installation

To play this level, extract the file "target.bsp" into
your \valve\maps directory and then start the game.
Start a deathmatch server and type "map target" or "changelevel
target" in the console. To test it out in single player mode
start Halflife with hl.exe -console -toconsole -dev +map target. 
This starts the map and lets you to try some stuff using the 
cheat codes.
