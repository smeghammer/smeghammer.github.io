* How to use this level *

Make a "maps" directory in ../quake2/baseq2 and copy the level into it.
In Quake2, bring up the console and type in "map wdwdm6".

To start a deathmatch game either start it as usual and type "map wdwdm6" in
the server's console or start a multiplayer server from Quake 2 and type
"map wdwdm6" in the console.
