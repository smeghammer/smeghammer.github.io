* How to use this wad *

You must use dmadds (included); 

Type: dmadds clintn30.wad clinton.wad  (You need to do this only once.  
                                        The resulting file will be 
                                        approx 2.2M.  It must be done
                                        in the dir where doom.wad is
                                        located!)
To run:

Type: doom -file clinton.wad clinsnd.wad
