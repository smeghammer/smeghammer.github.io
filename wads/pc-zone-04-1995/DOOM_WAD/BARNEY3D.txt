* How to use this wad *

You must use dmadds (not included); available at infant2.sphs.indiana.edu
/pub/doom/wad_edit/misc/dmadds1b.zip

Type: dmadds barney3d.wad barney.wad  (You need to do this only once.  The
                                       resulting file will be approx 2.2M)
Type: doom -f barney.wad brnysnd.wad
