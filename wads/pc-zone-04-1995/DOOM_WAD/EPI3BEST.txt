EPI3BEST.WAD
____________

This is a collection of the finest, custom made DOOM levels
found on the Internet and from various other places :)
EPI3BEST.WAD replaces E3M1 to E3M8 in DOOM v1.2 REGISTERED!
It will not work in the shareware version! To use this WAD, type

DOOM -FILE EPI3BEST.WAD

And doom will add the wad file to use at startup, using the levels
contained in this wad file instead of the default levels in DOOM.WAD
for episode 3 :)

Have fun & enjoy :)
