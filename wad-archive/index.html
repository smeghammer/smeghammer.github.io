Search in the WAD Archive stored at archive.org, implemented by querying the filenames.json file, stored locally, as provided with the data dump from 2022. Note that this builds links directly to the archive.org data. Many WAD Archive records also include other metadata (README's, screenshots, map images, TITLEPIC etc.) and this application will eventually allow viewing all of these if they are present.

Note that there is a delay on first search each time you visit (or hard-refresh) as the data is loaded when first search is performed, rathe rthan on page load.
